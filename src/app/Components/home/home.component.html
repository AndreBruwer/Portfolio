In what order did i develop my skills?

I started where everyone starts at - The Operating System. For me i started on Windows XP. Thinking back, there wasn't much you could do on it, but for me it was the most advanced piece of technology in the home. It being able to play simple games was a much needed bonus. Nevertheless, over the years I become more and more acquainted behind the logic that technology adhered to. Certain app and features are comonly found in certain places. There unspoken rules are everywhere and whenever app designers deviate from this norm, their users were sure to let their dissatisfaction known.


As the years passed and Operating systems got replaced by new one's, so too did my knowledge grow with it. By the time i entered highschool is could already do anything i wanted to with the apps contained within the Microsoft Office Package. From Word,Excel, all the way to Powerpoint. I could write whatever i wanted to, and make it look good in the process.

A few years later, we finally got internet at home and with most things online, you had to sign in with an email address to verify your identity. I opted to use Google's services and created my very first email address. I still use this email address to this very day. It was interesting back then when i created it because i wanted to use my name and surname. Oddly enough, my name and surname was already used by someone. So someone somewhere had either the same name and surname as me, or they were somehow creative enough to pick a name that exactly matches mine... Curious.

So with me having created a Google account, i was greeted with an ever-growing list of useful apps. I mastered them too as the years passed. I was overjoyed whenever they added a nice feature and disappointed whenever they replaces something nice with something that i wasn't happy with. But every time they broken a thing i liked, i just explored ways to achieve the same goal, given the different ways to achieve them.


I was also a very creative person, so once i discovered that i can visualize my creativity using Adobe Photoshop, i couldn't help but devour it. I made pieces that i liked so much, i made them my Desktop Wallpaper which impressed whomever saw them.

I had no prior experience with programming before i went to university, so i had to learn everything from the ground up. I quickly rose thorough the ranks as my practical skill were greatly appreciated by my peers who seeked advise.

We started with C# and Java. Java was primarily used to demonstrate Object Oriented Programming where C# were used to demonstrate how code could be affected by graphical components. We learned how to use simple SQL databases and queries to store data and we learned Github to make group work a lot more streamlined. I learned a lot of skills at university, skills that I don't even notice when I use them.

Once I graduated from university, my skills were put to the test. I soon learned that what was done at the workplace were much more complicated than whatever we did at university and that i was going to have to hit the ground running if i ever wanted to survive.


I was tasked with implementing a Database for the place i worked because the amount of data they were generating grew much faster than they previously anticipated. The simple Excel spreadsheets were no longer adequate for their operation. Thus i used my C# and SQL to make a simple database that was locally hosted on their servers.

Once that was complete i was tasked with improving their website as the people who previously maintained it left the company. Having no experience, i spent a few weeks getting used to HTML, CSS and Javascript as well as how they all fit together. I made a simple website that now that i look back, were barely passable but it served its purpose well.

The company then asked for a way to provide online courses given the technology they had to their disposal. This put my skills to an extreme test. and withing a few weeks, i had developed not only a system that they can easily add new online courses, I also replaced their previous website with one that looked and felt much better.

As time went on I ahd migrated the local SQL server and all of its data onto a AWS server that ran a PostgreSQL database. PostgreSQL is an improved version of SQL with a bunch of nice features as well as all the base features of SQL.

At the moment I am working on learning the Angular Framework, which will use and extend all of my previous website building skill.


IT is a journey of continuous development and as someone who loves to learn new things, it suits me like a glove.
